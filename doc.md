# NVIDIA Image Generation Explorer

A Python project that generates images using NVIDIA's AI API with the FLUX.2 Klein 4B model from Black Forest Labs.

## Project Structure

```
nvidia-explore/
├── .env                 # Environment variables (API token)
├── doc.md              # This documentation
├── generated_image.png # Output image (generated by the script)
├── imagemodel.py       # Main image generation script
```

## Requirements

- Python 3.x
- `requests` library
- `python-dotenv` library
- NVIDIA API token (NVIDIA AI Foundation Cloud)

## Setup

1. Install dependencies:
   ```bash
   pip install requests python-dotenv
   ```

2. Create a `.env` file with your NVIDIA API token:
   ```env
   IMAGE_API=your_nvidia_api_token_here
   ```

3. Get your NVIDIA API token from [NVIDIA NGC](https://org.ngc.nvidia.com/) or the NVIDIA AI Foundation Cloud platform.

## Usage

Run the script to generate an image:

```bash
python imagemodel.py
```

The script will:
1. Load the API token from the `.env` file
2. Send a request to NVIDIA's FLUX.2 Klein 4B model API
3. Decode the base64 image response
4. Save the generated image as `generated_image.png`

## Configuration

The [`payload`](imagemodel.py:16) object in `imagemodel.py` contains the generation settings:

| Parameter | Value | Description |
|-----------|-------|-------------|
| `prompt` | `"A group of 5 people taking a selfie..."` | Text description of the desired image |
| `width` | `1024` | Image width in pixels |
| `height` | `1024` | Image height in pixels |
| `seed` | `0` | Random seed for reproducibility (0 = random) |
| `steps` | `4` | Number of diffusion steps |

### Example: Modify the Prompt

To generate a different image, edit line 17 in [`imagemodel.py`](imagemodel.py:17):

```python
payload = {
    "prompt": "Your custom prompt here",
    "width": 1024,
    "height": 1024,
    "seed": 0,
    "steps": 4
}
```

## API Details

- **Endpoint**: `https://ai.api.nvidia.com/v1/genai/black-forest-labs/flux.2-klein-4b`
- **Method**: POST
- **Authentication**: Bearer token in Authorization header
- **Response**: JSON containing base64-encoded image in `artifacts[0].base64`

## Troubleshooting

- **401 Unauthorized**: Check that your `IMAGE_API` token is correct in `.env`
- **Connection errors**: Verify your internet connection and API endpoint availability
- **Image not saving**: Ensure write permissions in the project directory
